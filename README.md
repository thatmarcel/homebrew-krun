# Homebrew repository for libkrun and related tools

This is a Homebrew repository for:

- [libkrun](https://github.com/containers/libkrun)
- [krunvm](https://github.com/containers/krunvm)
- [krunkit](https://github.com/containers/krunkit)
- [krunai](https://github.com/slp/krunai)

## Supported platforms

The formulae and bottles on this repository only support macOS on Apple Silicon-based devices.

## How do I install these formulae?

```
brew tap libkrun/krun
brew trust libkrun/krun
brew install krunvm
brew install krunkit
brew install krunai
```
